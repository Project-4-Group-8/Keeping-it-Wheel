Model Description

Due to the price of the car being dependent on multiple varaibales (mileage, year, model), it was best to proceed using a Regression Model. However, which one would work the best? Below is the story of how our car prediction model came to be.

Regression Model Performance

To determine which model would provided the most accurate outcome for predicting car price, mutltiple models were tested such as Random Forest Regressor, Ridge Regression, LASSO, KNeighbors Regressor, and etc. The models were trained and tested using the year, make, model, miles, and trim of the cars. The following parameters were checked for each model to decide which would be the best to use:

  • Training Time
  • Train Score
  • Test Score

After reviewing the parameters, it was determined that the Random Forest Regression Model provided the best overall results. Further testing was performed using hyperparameters to see if the test score would improve for the model. However, it was ultimately decided that the peak perfromance of the model occurred without hyperparameters.
